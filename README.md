# 泉州 MBTI 神明答案之书

一个使用 Next.js、TypeScript 与 Tailwind CSS 制作的无图功能原型。用户可以选择 16 位 MBTI 神明，输入问题后抽取神谕答案，并通过茭杯结果确认答案是否成立。

## 本地预览

```bash
npm install
npm run dev
```

启动后在浏览器打开 `http://localhost:3000`。

## 可用脚本

- `npm run dev`：启动开发服务器。
- `npm run build`：构建生产版本。
- `npm run start`：启动生产服务器。
- `npm run lint`：运行 Next.js ESLint 检查。
